What is Landlab?

Landlab is a Python-based modeling environment that allows scientists and students to build numerical landscape models. Designed for disciplines that quantify earth surface dynamics such as geomorphology, hydrology, glaciology, and stratigraphy, it can also be used in related fields.

Landlab provides components to compute flows (such as water, sediment, glacial ice, volcanic material, or landslide debris) across a gridded terrain. With its robust, reusable components, Landlab allows scientists to quickly build landscape model experiments and compute mass balance across scales.

Landlab is described in more detail by Hobley et al. in the 2017 paper Creative Computing with Landlab.

Examples

The following models can easily be built with Landlab's current capabilities:

  • A landscape evolution model using linear diffusion and the stream-power model
  • A model that explores the flexural response to the growth and recession of glaciers
  • An ecohydrology model in which vegetation on two sides of a valley grows and dies in response to stochastic storms and solar forcing throughout the year
  • A model that routes hydrographs across a watershed based on rainfall inputs

    • Installing Landlab

    To install Landlab, you will need a Python distribution that includes

  • Python 2.6+ or 3.5+
  • NumPy 1.9 or greater
  • SciPy 0.14 or greater
  • netCDF4 (will run without, but recommended)
  • Cython (only required if building Landlab from source code)
  • an up-to-date Python package manager (such as PyPI, aka "pip")
  • a compiler that Python can recognise (through the Xcode app on a mac)

    • To fulfill these requirements we recommend a preassembled scientific Python distribution such as Continuum IO’s Anaconda. Follow this link for instructions on installing your Python distribution.

    Once your Python distribution is complete and up to date, install Landlab.

    Installation Options

  • Using Anaconda - installation of a pre-compiled Landlab release - it's quick but you will not be able to access and edit the code
  • Developer Installation - forking Landlab's github repository - preferred option if you want to modify code and commit back to Landlab

    • Installing Landlab Using Anaconda

    Open a terminal (or, on a PC, the command prompt—make sure it is a true DOS "cmd" prompt, not, e.g., Cygwin) and type the following:

    > conda install landlab -c landlab -c conda-forge

    If you would like more detailed instructions for installing Landlab with Anaconda's Python distribution, follow this link.

    Updating Landlab

    As Landlab is still relatively early in its development cycle, the code will update fairly often with frequent new releases. To take advantage of new features and new library additions, we recommend you update Landlab regularly.

    Whenever you update Landlab, we strongly recommend you also update your Python package! For Anaconda, use the conda package manager from a command prompt. [Remember, on a PC, do all this in a Command Prompt window or through Anaconda; do not use Cygwin.]

    > conda update --all

    From Canopy, use the GUI to update all the available new packages listed.

    If you installed Landlab through the instructions on this page, simply use pip to update, like so:

    > pip uninstall landlab

    > pip install landlab

    However, if you are running an Anaconda distribution, we now recommend you uninstall with pip and replace with a conda install. Simply do this:

    > pip uninstall landlab

    > conda install landlab -c landlab -

    Once you've done this, you will be able to update landlab along with the rest of your conda packages:

    > conda update --all -c landlab
